# Personal Portfolio
 
### Built using 
    - Django
    - PostgreSQL 10
    - Jinja
    - Bootstrap

### Implements
    - A Home Page
    - Email Module
    - Rest API
    - DB Models
    - Image Upload Module

### Important Commands
```
python3 manage.py runserver

python3 manage.py collectstatic

python3 manage.py makemigration

python3 manage.py migrate
```
