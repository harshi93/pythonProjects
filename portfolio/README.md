# Personal Portfolio
 
### Built using 
    - Django
    - PostgreSQL 10
    - Jinja
    - Bootstrap
    - Django Widget Tweaks

### Implements
    - A Home Page
    - Email Module
    - Rest API
    - DB Models
    - Image Upload Module For Admin Users
    - Endpoints
        - /
        - /jobs/<int:id>
        - /discuss

### Important Commands

- Install packages
    ```
    pip install -r requirements.txt
    ```

- Install DB Models
    ```
    python3 manage.py makemigration

    python3 manage.py migrate

    ```

- Start server
    ```
    python3 manage.py runserver
    ```
